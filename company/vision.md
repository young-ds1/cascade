# Vision — 为什么存在

## 核心信念

AI 编程工具的能力不应该被 API 价格和协议壁垒锁死。

中国开发者应该能用任何国产模型、驱动任何 AI 编程工具、花最少的钱、写最好的代码。

## 我们相信的事

1. **协议不应该是壁垒**。Responses API vs Chat Completions 只是格式差异，不应该是付费的理由。
2. **Key 应该留在用户手里**。本地跑、本地翻译、不作中转。
3. **开源是最高效的分发**。MIT 协议，零门槛，fork 即拥有。
4. **中文优先不是翻译**。是文档、面板、设计决策都从中国开发者的场景出发。

## 10 年愿景

成为 AI 编程工具和国产模型之间的标准翻译层。

不管什么工具、什么模型，一行命令，Cascade 让它们通话。
